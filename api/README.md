### Instalação

Para instalar os pacotes utilizados pela API execute o comando abaixo:

```bash
$ pipenv install
```

### Execução

Para executar é necessário ativar o ambiente de desenvolvimento:

```bash
$ pipenv shell
```

Agora, é só executar o servidor:

```bash
python server.py
```
